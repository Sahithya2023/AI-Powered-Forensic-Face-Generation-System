# AI-Powered Forensic Face Generation System (Prototype)

Prototype system that generates realistic suspect faces from witness descriptions using Stable Diffusion and Streamlit.

## Features
- Face generation from witness input
- Confidence-based feature weighting
- Facial attribute refinement
- Interactive UI
- Multiple face variations generation

## Tech Stack
- Python
- PyTorch
- Streamlit
- Stable Diffusion
- Diffusers

## Run

```bash
streamlit run forensic_face_app_v2.py
